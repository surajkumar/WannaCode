# WannaCode
An ultra light IDE for Java programming.

## Pre-requisites
* You will need to place the eclipse-jdt folder into the root of this project `WannaCode`. To download Eclipse JDT: https://download.eclipse.org/jdtls/snapshots/jdt-language-server-1.51.0-202509042040.tar.gz
* Python is needed for Eclipse JDT to start
